# MediSign - AI-Powered Handwritten Prescription Translator

## overview
MediSign is an AI-powered healthcare solution that converts handwritten medical prescriptions into readable digital text using Optical Character Recognition (OCR) and Natural Language Processing (NLP).

## Problem Statement
Handwritten prescriptions are often difficult to interpret, leading to medication errors and delays in healthcare processes.

## Features
- Handwritten prescription text extraction
- OCR-based text recognition
- Medication name identification
- Image preprocessing for improved text quality
- Digital prescription generation

## Tech Stack
- Python
- OpenCV
- Tesseract OCR
- TrOCR
- NumPy
- Pandas

## Dataset
- Medical prescription datasets from Kaggle

## Project Workflow
1. Image Input
2. Image Preprocessing
3. OCR Text Extraction
4. Text Interpretation
5. Digital Output Generation

## Results
- Successfully extracted handwritten prescription text
- Improved readability through image enhancement techniques
